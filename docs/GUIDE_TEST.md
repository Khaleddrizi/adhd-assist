# دليل التجربة — سيناريو كامل

المهم: ولي الأمر يجب أن يسجّل أولاً، ثم الطبيب ينشئ المريض ويربطه ببريد ولي الأمر.

## الخطوات

### 1. تشغيل المشروع
```bash
# Terminal 1: Backend
source venv/bin/activate
python run.py

# Terminal 2: Frontend
cd frontend && npm run dev
```

### 2. إنشاء حساب ولي الأمر أولاً
1. افتح http://localhost:3000/register
2. أنشئ حساب: نوع **Parent** (ولي الأمر)
   - مثال: `parent@test.com` / `password123`
3. بعد التسجيل سيتم توجيهك لصفحة ولي الأمر

### 3. إنشاء حساب الطبيب
1. سجّل خروج أو افتح نافذة خاصة (incognito)
2. اذهب إلى `/register`
3. أنشئ حساب: نوع **Orthophoniste** (الطبيب)
   - مثال: `dr@test.com` / `password123`
4. بعد التسجيل سيتم توجيهك لصفحة الطبيب

### 4. الطبيب يسجّل المريض (الطفل)
1. سجّل دخول بحساب الطبيب
2. اضغط **إضافة مريض** (Add patient)
3. أدخل:
   - الاسم (مثلاً: Ahmed Test)
   - العمر
   - التشخيص
   - **البريد الإلكتروني لولي الأمر:** `parent@test.com` (نفس إيميل ولي الأمر المسجّل)
4. احفظ — سيظهر **كود Alexa** (مثل: A1B2C3D4)
5. **احفظ هذا الكود** — المريض سيستخدمه في Alexa

### 5. تجربة Alexa
1. استخدم محاكي Alexa أو جهاز Echo
2. قل: **"Link"** ثم الكود، مثلاً: **"Link A1B2C3D4"**
3. قل: **"Give me a quiz"**
4. أجب على الأسئلة (A, B, C)
5. قل: **"End quiz"** للحصول على النتيجة

### 6. عرض النتائج
- **صفحة الطبيب** (`/orthophoniste`): قائمة المرضى + إحصائيات + جلسات
- **صفحة ولي الأمر** (`/dashboard`): قائمة الأطفال + إحصائيات + جلسات
